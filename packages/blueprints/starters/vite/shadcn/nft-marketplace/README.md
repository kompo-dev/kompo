# NFT Marketplace

NFT Marketplace with buying, selling and minting features.

## Features

- NFT Listing & Browsing
- Buying and Selling NFTs
- NFT Minting functionality
- Wallet connection integration
- Modular architecture with Hexagonal Domain and Adapters

## Stack

- **Frontend**: React + Vite
- **Design System**: Shadcn UI (Tailwind CSS v4)
- **Web3**: Viem / Wagmi
- **Database**: Drizzle ORM (via adapter)
- **Architecture**: Domain-Driven Design (Hexagonal)
