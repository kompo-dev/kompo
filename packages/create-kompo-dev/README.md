# Create Kompo Dev

The quickest way to get started with Kompo, a modern Web3 framework for TypeScript.

## Quick Start

To create a new project, run:

```bash
pnpm create kompo-dev
```

_Alternatively, you can use npm, yarn or bun:_

```bash
npm init kompo-dev
# or
yarn create kompo-dev
# or
bun create kompo-dev
```

## Documentation

For full documentation, visit [kompo.dev](https://kompo.dev/docs).

## Why Kompo?

Kompo provides a structured, type-safe environment for building complex Web3 applications with ease. It handles the boilerplate so you can focus on your business logic.
